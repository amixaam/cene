# CENE
School project - A website for event ticket purchase

made in a week.
## Includes: 
* Stripe integration
* Stripe Webhooks
* PDF/QR code for boughts tickets
* Seat selector when buying
* Admin panel for events
* Safe Login/Register
* Filter events by genre and or search
* Reviews for ticket owners
* Custom ticket types
* Custom genres
* Custom age ratings
* Custom seat amount
